# 🎤 Text To Speech & Speech To Text

A simple **C# Windows Forms** application that provides both **Text-to-Speech (TTS)** and **Speech-to-Text (STT)** features.

This project uses:

- **System.Speech.Synthesis** for reading text aloud  
- **NAudio** for recording audio  
- **OpenAI Whisper API** for transcription  

---

## 🚀 Features

- 🔊 Convert typed text into speech  
- 🎙️ Record audio from microphone  
- 🧠 Transcribe audio into text using Whisper  
- 💬 Display results in real-time  
- 🧩 Simple and clean UI  

---

## 🛠️ Technologies Used

- **C#**
- **.NET WinForms**
- **NAudio**
- **System.Speech.Synthesis**
- **OpenAI Whisper API**
